---
name: fluid-theme
description: Restricts sizing to clamp functions and enforces high-end semantic token balances.
---
- Never use fixed pixel variables for padding or typography. Always use fluid scales like `clamp(1rem, 2vw, 3rem)`.
- Enforce strict dark-mode luxury scales: base surface (#0B0B0C), alt surface (#141416), low-opacity thin border (rgba(255,255,255,0.06)).
